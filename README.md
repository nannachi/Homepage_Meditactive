# 🌿 Meditactive – Landing Page

Questo progetto è una home page realizzata in HTML e CSS, pensata per un'app dedicata alla meditazione e alla crescita personale.

## ✨ Descrizione

Meditactive è un concept di piattaforma che aiuta gli utenti a:
- ritrovare equilibrio interiore
- migliorare la concentrazione
- costruire abitudini quotidiane sane

La pagina è stata progettata con un focus su:
- semplicità
- leggibilità
- layout pulito
- coerenza visiva

## 🛠️ Tecnologie utilizzate

- HTML5  
- CSS3  
- Google Fonts (Urbanist)

## 🎯 Obiettivi del progetto

- Creare una landing page strutturata da zero  
- Applicare principi base di UI/UX  
- Gestire layout e spaziature in modo professionale  
- Utilizzare Git e GitHub per versionamento del codice
